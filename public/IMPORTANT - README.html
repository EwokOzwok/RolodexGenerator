üåü Method 1: Deploy to shinyapps.io (Easiest!)

This is the fastest and easiest way to get your app online - no technical skills required!

1Get Your Data File Ready

Find your CSV file (the spreadsheet with all your data) and rename it to exactly rolodex.csv

Put this file in the same folder as all your other Rolodex App files.

üí° Tip: Make sure the CSV has the same columns as the original file you used.

2Install R and RStudio

If you don't already have them:

  1. Download and install R: https://cran.r-project.org/
  2. Download and install RStudio: https://posit.co/download/rstudio-desktop/

3Install Required Packages

Open RStudio and run these commands in the Console (bottom-left panel):

install.packages(c("shiny", "shinyMobile", "rsconnect"), dependencies = T)

Press Enter after pasting. This might take a few minutes.

4Create a Free shinyapps.io Account

  1. Go to shinyapps.io
  2. Sign up for a free account
  3. Verify your email address

5Connect RStudio to Your shinyapps.io Account

You'll need to connect RStudio to your shinyapps.io account just once. Here's how:

  1. Go to your shinyapps.io dashboard
  2. Click on your name in the top-right corner, then select Tokens
  3. Click Show next to "Token" and then Show Secret
  4. Click Copy to clipboard
  5. Back in RStudio, paste this into the Console (the bottom-left panel) and press Enter
üí° Need more detailed help? Check out this complete step-by-step guide: Deploying to shinyapps.io

6Publish Your App! üöÄ

This is the exciting part - and it's super easy!

  1. Make sure your app.R file is open in RStudio
  2. Look for a blue "Publish" button in the top-right corner of the code editor
  3. Click it!
  4. A window will pop up - just click Publish again
  5. Wait a minute or two while your app uploads (you'll see a progress bar)
üéâ Done! Your app is now live on the internet! You'll get a link that looks like https://yourname.shinyapps.io/rolodex/ that you can share with anyone!

üåê Method 2: Embed Your App on Your Website

Once your app is deployed (using shinyapps.io or Docker), you can embed it directly into your own website using an iframe.

1Get the Embed Code

Open the embed-app.txt file in your Rolodex App directory. You'll see HTML code that looks like this:

<iframe src="https://yourname.shinyapps.io/yourAppsName" 
        width="100%" 
        height="800" 
        frameborder="0">
</iframe>
‚ö†Ô∏è Important: Edit the code and replace https://yourname.shinyapps.io/yourAppsName with the actual web address where your app is hosted.

2Add to Your Website

  1. Create a new page on your website (or edit an existing one)
  2. Paste ONLY the HTML code (the iframe code above) into the empty web page
  3. Save and publish your page
üéä That's it! Your Rolodex App is now embedded on your website and visitors can use it directly from your page!

üì± Method 3: Enable Progressive Web App (PWA) Features

Want users to be able to install your app on their phones and use it like a native app? Follow these steps to enable PWA features!

1Update the Manifest File

  1. Navigate to the www folder inside your main Rolodex App directory
  2. Open the manifest.webmanifest file in a text editor
  3. Find the line that says: "start_url": "/"
  4. Change it to: "start_url": "https://yourname.shinyapps.io/yourAppsName"
  5. Save the file
‚ö†Ô∏è Important: Replace https://yourname.shinyapps.io/yourAppsName with your actual app URL from shinyapps.io.

2Redeploy Your App

After updating the manifest.webmanifest file, redeploy your app to shinyapps.io following the same steps from Method 1, Step 6.

üéâ Done! Your app now supports PWA features! Users can install it on their mobile devices and use it like a native app with offline capabilities!

üê≥ Method 4: Deploy with Docker (Advanced)

This method gives you more control and allows you to run the app on your own computer or server. It's a bit more technical, but we'll guide you through it!

üëã First time using the command line? No worries! Just follow each step carefully. Whenever you see a command in a gray box, copy it, paste it into your window, and press Enter on your keyboard to run it. Easy! üí™

1Get Your Data File Ready

Find your CSV file (the spreadsheet with all your data) and rename it to exactly rolodex.csv

Put this file in the same folder as all your other Rolodex App files.

üí° Tip: Make sure the CSV has the same columns as the original file you used.

2Download Docker Desktop

Docker is like a self-contained box that has everything your app needs to run.

Download it here: Docker Desktop

Choose the Windows or Mac version depending on your computer.

üçé Mac Users: You need to know if your Mac has an Intel chip or an Apple Silicon (M1/M2/M3) chip before downloading Docker Desktop.
  1. Click the Apple logo in the top-left corner
  2. Select About This Mac
  3. Look for the line that says Processor or Chip
  4. If it says "Intel" ‚Üí download the Intel (x86) version of Docker Desktop
  5. If it says "Apple M1/M2/M3" ‚Üí download the Apple Silicon version

3Install Docker Desktop

Double-click the file you just downloaded and follow the steps. After it installs, open Docker Desktop and wait for the little whale icon üê≥ to appear ‚Äì that means it's ready.

4Open Your Command Window

ü™ü For Windows Users:

  1. Click the Start menu
  2. Type powershell and press Enter
  3. Click on Windows PowerShell when it appears

üçé For Mac Users:

  1. Press Command + Space
  2. Type terminal
  3. Press Enter

5Navigate to Your App Folder

Use the cd command (means "go to folder") to point your terminal to where your Rolodex App files are.

Example for Windows:

cd "C:\Users\YourName\Desktop\Rolodex App"

Example for Mac:

cd "/Users/YourName/Desktop/Rolodex App"

Press Enter after typing or pasting the command.

6Build and Start Your App

Now let's build and run the app. Copy the commands below depending on your computer:

ü™ü Windows or Intel-based Mac

First build (wait 2‚Äì3 minutes):

docker build -f Dockerfile.txt -t rolodex .

Then run:

docker run -d --restart=always -p 1222:1222 --name RolodexApp rolodex

üçé Apple Silicon (M1/M2/M3) Mac

First build (forces compatibility mode):

docker build --platform=linux/amd64 -f Dockerfile.txt -t rolodex .

Then run:

docker run -d --restart=always -p 1222:1222 --name RolodexApp rolodex
üéâ That's it! Your app is now running on your computer.

7Open Your App in a Browser

Open your browser and go to:

http://localhost:1222

If Docker Desktop is running, your app will load! üéä
