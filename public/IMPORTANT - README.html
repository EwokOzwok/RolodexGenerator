Rolodex App Deployment Guide

This guide will walk you through the steps needed to deploy and run the Rolodex App on your local machine or in the cloud. Follow each step carefully to ensure successful deployment.

Step 1: Prepare Your Data

  • Place your resources .csv file in the main project directory.
  • The filename must be named exactly rolodex.csv.
  • The CSV file must match the same format and structure as the one you uploaded originally.

Step 2: Download Docker Desktop

The Rolodex App is containerized and requires Docker Desktop to run.

  • Download the Windows or Mac version of Docker Desktop: https://www.docker.com/products/docker-desktop/

Step 3: Install Docker Desktop

Run the installer you downloaded in Step 2 and complete the installation process on your computer.

Step 4: Build the Docker Container

  1. Open Command Prompt (Windows) or Terminal (Mac).
  2. Navigate to the project folder using the cd command.
    Example: 
    cd "C:/Users/Admin/Desktop/Rolodex App"
  3. Run the following commands to build and run the app container: 
    docker build -f Dockerfile.txt -t rolodex .
docker run -d --restart=always -p 1222:1222 --name RolodexApp rolodex

Step 5: Access Your App

Once the container is running, open your browser and navigate to:

  • http://localhost:1222
  • or http://127.0.0.1:1222

Step 6: Advanced - Deploy to the Cloud

To make your app accessible online, you can push your Docker image to Docker Hub and then deploy it using a cloud service such as Google Cloud Run.

  • Requires a free Docker Hub account.
  • Once pushed, you can deploy the image to hosting providers that support Docker.
